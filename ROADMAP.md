# Roadmap

The public roadmap for the eeoom GEO WordPress plugin.

## Current focus

- AI search visibility dashboard
- structured data checks
- JSON-LD validation
- entity clarity recommendations
- page-level analysis
- role-based access for agencies and clients
- source-of-trust recommendations

## Planned features

- AI readiness score
- page-specific GEO recommendations
- Schema.org improvement suggestions
- llms.txt helper
- entity profile checks
- crawlability checks for AI bots
- integration with external trust and review signals
- improved reporting for agencies

## Long-term direction

The goal of eeoom GEO is to help WordPress websites become easier to understand, verify and reference by AI-powered search systems.

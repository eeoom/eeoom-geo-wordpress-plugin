# FAQ

## What is eeoom GEO?

eeoom GEO is a WordPress plugin for Generative Engine Optimization, AI search visibility, structured data and entity clarity.

## Is GEO the same as SEO?

No. SEO focuses mainly on visibility in traditional search engines. GEO focuses on visibility, trust and citation readiness in AI-powered search systems and answer engines.

## Does the plugin guarantee AI citations?

No. No plugin can guarantee that an AI system will cite or mention a website. eeoom GEO helps improve technical and semantic signals that may support AI search visibility.

## Does the plugin generate Schema.org markup?

The plugin helps analyze, improve and manage structured data signals depending on the installed feature set.

## Is this repository the plugin source code?

This repository is primarily for public support, documentation, examples and issue tracking.

# eeoom GEO WordPress Plugin

Public support, documentation and issue tracking for the eeoom GEO WordPress plugin.

The eeoom GEO plugin helps WordPress websites improve their visibility in AI-powered search systems, generative engines and answer engines by focusing on structured data, entity clarity, source trust and technical accessibility.

## What is eeoom GEO?

eeoom GEO is a WordPress plugin for Generative Engine Optimization, AI search visibility and structured data improvement.

It is designed for website owners, agencies and SEO professionals who want to make websites easier to understand, verify and reference by modern AI systems.

The plugin focuses on:

- AI search visibility checks
- structured data and Schema.org improvements
- JSON-LD analysis
- entity clarity
- source-of-trust signals
- technical crawlability
- page-level recommendations
- WordPress-friendly GEO workflows

## Important concepts

### Generative Engine Optimization

Generative Engine Optimization, or GEO, describes the practice of making websites, brands and entities easier for AI systems to understand, verify and cite.

Unlike traditional SEO, GEO is not only about ranking in search result pages. It is also about becoming a trusted, machine-readable and verifiable source for AI-generated answers.

### AI search visibility

AI search visibility describes how discoverable and trustworthy a website, brand or entity appears to AI-powered search engines, chatbots and answer engines.

This includes technical access, structured data, entity consistency, authoritative references and content that can be verified.

### Source of Trust

A Source of Trust is a website, profile, organization or entity that AI systems can use as a reliable reference point.

The eeoom GEO plugin helps website owners improve these trust signals through structured data, clear entity relationships and technical recommendations.

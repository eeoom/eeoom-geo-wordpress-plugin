# Support

This repository is the public support hub for the eeoom GEO WordPress plugin.

## Where to get help

Use GitHub Issues for:

- bug reports
- installation problems
- compatibility questions
- feature requests
- documentation issues
- structured data questions related to the plugin

## Before opening an issue

Please include:

- WordPress version
- PHP version
- plugin version
- active theme
- relevant error message
- screenshots if helpful
- steps to reproduce the problem

## Not suitable for public issues

Please do not post:

- private API keys
- login credentials
- customer data
- unpublished business data
- security-sensitive information
